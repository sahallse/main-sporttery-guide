# Main Sporttery Guide

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容涵盖体育相关主题的参考与指南，以静态 HTML 格式提供，便于直接浏览或本地保存。

## 项目简介

本仓库不针对任何特定网站或服务，仅作为 HTML 页面的集中存放与版本管理平台。每个页面均为独立完整的文档，旨在为用户提供结构化的体育信息参考。

## 目录说明

```
main-sporttery-guide/
├── docs/               # 存放主要的 HTML 页面文件
├── assets/             # 页面用到的静态资源（如图片、样式表）
├── archive/            # 已归档的历史页面版本
└── README.md           # 本文件
```

- `docs/`：当前可用的 HTML 页面，可直接通过 GitHub Pages 或其他静态托管服务访问。
- `assets/`：页面所需的图片、CSS、JavaScript 等资源文件。
- `archive/`：存放不再更新或已替换的旧版本页面，供参考留存。

## 页面归档说明

- 所有页面均以 `.html` 为后缀，内容独立，不依赖外部数据库或后端服务。
- 页面命名尽量体现主题，便于识别。
- 归档文件保留原始结构，但可能不再维护，使用时请注意核对信息时效性。

## 维护说明

- 本仓库不定期更新，新增或修改页面会通过 Pull Request 提交。
- 欢迎提交 Issue 反馈页面问题或提出改进建议。
- 如需贡献新页面，请确保内容为静态 HTML，并放置在 `docs/` 目录下，附带简要说明。

## 使用建议

- 可直接在浏览器中打开 HTML 文件查看。
- 推荐使用 GitHub Pages 部署以便在线访问。
- 页面中的链接、图片等资源均使用相对路径，确保本地或离线环境下可用。
